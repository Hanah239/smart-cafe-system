# Smart Café Self-Order System

## Overview

This project is a desktop-based café ordering system built using Python and MySQL. It simulates a self-order kiosk where users can select items, manage a cart, and generate a bill.

## Features

* Item selection and customization
* Cart management and billing with VAT calculation
* Basic inventory handling
* Graphical user interface using Tkinter

## Tools & Technologies

* Python
* MySQL
* Tkinter

## What I Worked On

* Implementing core functionality such as order processing and billing logic
* Understanding how the system interacts with a database for storing and retrieving data
* Exploring event-driven programming through user interactions in the interface

## Key Learning

This project helped me understand how applications combine user interfaces, backend logic, and databases to handle real-time data processing.
